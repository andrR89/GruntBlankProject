Instalação

Passos
  • 1   Clonar o Projeto

  • Abrir o CMD e executar o seguinte comando: 
    git clone ssh://repos.nexxera.com/home/repos/git/teste/GruntBlankProject
  • 2  Executar grunt-install.bat

  • Ainda no cmd Navegar ate a pasta "config" do projeto e executar o comando abaixo: 
    cd $PROJECT_HOME/config
grunt-install.bat
  • 3  Rodar a distribuição
  • Ainda na pasta config, executar a task do grunt no cmd 
    grunt execute-dist
                
grunt qa // Executa os testes no projeto e roda o JsHint
grunt dist // Gera a distribuição do Projeto
grunt execute-dist // Gera a distribuição do Projeto e Roda o mesmo num Servidor Web
